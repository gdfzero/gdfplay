# Requirements
- SDK Minimum: Android SDK 24
- SDK 컴파일: Android SDK 33
- NDK :  Android NDK 21.4
- laguege: Java/Kotlin
- x86 기반 android simulator에서는 정상 동작하지 않습니다.(실제 device에서 테스트 요망)
- tensorflow lite제안으로 인해 SDK는 arm64-v8a and armeabi-v7a기반 모바일에서만 작동합니다.
